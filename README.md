# Kiwi
Kiwi is a high-performance, modular, client-side pathfinding bot for Minecraft 1.21+ (Fabric).

> [!IMPORTANT]
> Kiwi is in a very early state. Use with caution.

## Features
- Fast [**Theta***](https://en.wikipedia.org/wiki/Theta*) pathfinding
- Block breaking
- Block placing
- [Goals](#Goals)

## Goals
- **GoalXYZ** a specific xyz coordinate
- **GoalNear** get into a range of an xyz coordinate
- **GoalXZ** a specific xz coordinate
